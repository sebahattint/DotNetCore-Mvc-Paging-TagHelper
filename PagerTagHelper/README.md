# .Net Core 2.1 Mvc Paging Tag Helper Example

<h2>Use in cshtml</h2>

```
<mytags-pager-nav 
  page-size="@Model.PageSize" 
  page-count="@Model.PageCount" 
  current-page="@Model.CurrentPageNo">
</mytags-pager-nav>
```
